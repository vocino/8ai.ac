Zed is a high-performance code editor built from the ground up with AI and collaboration at its core. It features a built-in AI assistant that supports inline code generation, editing, and conversation with multiple LLM providers including Anthropic, OpenAI, and local models via Ollama. Zed's GPU-accelerated architecture makes it one of the fastest editors available, and its real-time collaboration features allow multiple developers to work in the same file simultaneously.
