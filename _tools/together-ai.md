Together AI is a cloud platform for running, fine-tuning, and deploying open-source AI models. It offers one of the broadest selections of open-source LLMs — including the full Llama 3 family, Mistral variants, and Stable Diffusion models — at competitive pricing. Together AI's fine-tuning API lets developers customize models on proprietary data, and its API is designed to be OpenAI-compatible for easy migration.
