What is LlamaIndex? LlamaIndex is the leading open-source framework for building retrieval-augmented generation (RAG) applications. It provides data connectors, indexing strategies, and query engines that make it easy to connect LLMs to custom data sources. LlamaIndex supports advanced RAG patterns, agent workflows, and integrates with all major LLM providers. LlamaCloud offers a managed production service for enterprise deployments.
