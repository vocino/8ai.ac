Clipdrop is an AI image editing toolkit developed by Stability AI that provides a suite of tools for quickly transforming and enhancing photos. Core features include background removal, object and text removal, AI upscaling, and image generation using Stable Diffusion. It is available as a web app, API, and mobile app, with a developer API suitable for product integrations.
